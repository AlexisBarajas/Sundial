<h2 name="Installation">Installation</h2>

- Clone the repository

  ```sh
  git clone https://github.com/AlexisBarajas/Sundial
  ```
- Install the packages

  ```sh
  npm install
  ```
- Run the project
  ```sh
  npm start
  ```
